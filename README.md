# chronomy-api

API backend for [mbrad26/final-project-front-end](https://github.com/mbrad26/final-project-front-end)

## Setup

```bash
$ bundle install
$ rails db:create
$ rails db:migrate
$ rails s -p 3001
```

## Testing

To run RSpec (with coverage) followed by Rubocop run script:  

```bash
$ ./test/tdd.test
```
